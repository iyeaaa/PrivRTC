title 화면 영역 세밀 지정

    participant User
    participant WebWorker
    participant Helper

    User->>User: "영역 선택" 버튼 클릭

    User->>User: 슬라이더 조작 (top, bottom, left, right 값 변경)
    User->>User: applySliderValues() -> croppingScreen(t, b, l, r) 호출
    User->>User: MediaStreamTrackProcessor/Generator 생성
    User->>WebWorker: 'crop' 메시지 전송 (readable, writable 스트림 및 crop 값 전달)
    WebWorker->>WebWorker: onmessage 이벤트 수신
    WebWorker->>WebWorker: readable 스트림에서 프레임 읽기
    WebWorker->>WebWorker: transform() 함수 실행 (crop 값 적용하여 프레임 자르기)
    WebWorker->>WebWorker: 잘린 프레임(newFrame)을 writable 스트림에 쓰기
    User->>User: Generator 트랙에서 변환된 스트림 생성
    User->>User: screenVideo.srcObject = new MediaStream([generator]) (로컬 미리보기 업데이트)
    User->>User: PeerConnection의 비디오 트랙 교체 (replaceTrack(generator))
    Note over User,Helper: 변경된 스트림이 Helper에게 전송됨 (기존 track 이벤트 핸들러 재사용)
    Helper->>Helper: PeerConnection 'track' 이벤트 (또는 기존 스트림 업데이트)
    Helper->>Helper: Helper 화면에 잘린 User 화면 표시
